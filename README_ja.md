## 特徴

- ピンクのツインブレイド、白いベレー帽、やさしい瞳を持つ Q 版 3D トイ風の Nikki。
- *Bloom Poetry* のピンクチェックドレス、リボン、レース、花のディテールを小さな Codex ペットとして読みやすく整理。
- Codex 仕様: `1536x1872`、8 列 x 9 行、1 コマ `192x208`。
- `idle`、`running-right`、`running-left`、`waving`、`jumping`、`failed`、`waiting`、`running`、`review` の 9 状態を収録。
- RGBA、透明背景、未使用セル透明、透明ピクセルの RGB 残留なしで検証済み。

## インストール

Releases から `nikki-bloom-codex-pet-v1.0.0.zip` をダウンロードし、次の 2 ファイルを Codex のカスタムペットフォルダへコピーします。

```text
pet.json
spritesheet.webp
```

```text
$HOME/.codex/pets/nikki-bloom/
├── pet.json
└── spritesheet.webp
```

利用可能なファイル:

- [dist/pet.json](dist/pet.json)
- [dist/spritesheet.webp](dist/spritesheet.webp)

Codex を再起動し、カスタムペット一覧から **Nikki Bloom** を選択してください。
